# SET 9, A1

Содержимое папки:

- `A1_experiment.cpp` — генератор строк, тестер и реализации сортировок для эксперимента.
- `A1_report.tex` — LaTeX-заготовка отчета по аналитической задаче.
- `quick_results.csv` — быстрый проверочный прогон, если запустить команду ниже.
- `results.csv` — полный прогон, если нужен весь набор измерений.

Быстрая проверка:

```bash
g++ -std=c++17 -O2 A1_experiment.cpp -o A1_experiment
./A1_experiment --quick > quick_results.csv
```

Полный эксперимент:

```bash
./A1_experiment > results.csv
```

В отчете нужно заменить поля `ID посылки` и ссылку на публичный репозиторий после отправки решений в Codeforces и публикации CSV.
